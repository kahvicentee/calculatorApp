## 🔧 Built with
- HTML5
- CSS3
- JavaScript (Vanilla)

## ⚙️ Features
- Perform mathematical operations like addition, subtraction, multiplication, and division;
- Custom-built calculation logic implemented without using eval();
- Full support for both keyboard and button inputs;
- Switch between different color themes based on user preference;
- Reset and delete functionality;
- Responsive layout for different screen sizes.
